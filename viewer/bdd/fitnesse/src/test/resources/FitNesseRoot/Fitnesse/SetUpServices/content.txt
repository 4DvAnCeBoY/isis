!2 Description
Registers the services, from which the rest of the domain classes are discovered.  Part of the bootstrapping of the test framework itself, typically referenced in .BootstrapNakedObjects included in the test suite's setup page.

!2 Arguments
 * none

!2 Columns
 * class name
   * the fully qualified name of the service implementation
 * alias=
   * assigns an alias to the service instance
   * can also use 'alias as'

!2 Example Usage
!|set up services|
|class name|alias=|
|com.mycompany.myapp.service.claim.ClaimRepositoryInMemory|claims|
|com.mycompany.myapp.service.claim.EmployeeRepositoryInMemory|claims|
