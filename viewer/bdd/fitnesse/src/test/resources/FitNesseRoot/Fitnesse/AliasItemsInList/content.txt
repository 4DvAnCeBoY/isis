!2 Description
Provide an alias to items in list, which are presumed to exist.  See also CheckList.

!2 Arguments
 * list alias

!2 Columns
 * property name(s)
 * alias as
   * can also use 'alias='

!2 Example Usage

!|alias items in list|list1|
|title|alias as|
|Tom Brown|tomEmployee|
