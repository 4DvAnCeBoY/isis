Properties

The follow details what properties are can be specified in the properties files (by default nakedobjects.properties in the working directory).

To hide the splash screen

nakedobjects.nosplash

To set the local (as used by date and times)

nakedobjects.locale=locale code

For example the following lines would be used for setting the locale for Japan, England, or France 

nakedobjects.locale=ja_JP
nakedobjects.locale=en_GB
nakedobjects.locale=fr_FR
