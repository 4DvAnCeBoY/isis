Naked Object Context/Container

NakedContainer is an interface that give clients access to:

  1. the class manager
  2. the object manager
  3. utitlities to create transparent and persistent objects
  4. the user of the system, if there is one

Exploration applications, XATs, client viewers, and servers all provide a reference to such a container.  This reference is assigned to each naked object by calling setContainer().  This method should be implemented such that it stores this context, for later use.
